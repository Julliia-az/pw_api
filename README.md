# 🌤️ Site de Clima

Aplicação web que exibe informações climáticas em tempo real a partir de uma API externa.

## 🚀 Demo
🔗 Acesse o projeto: https://seu-link-github-pages.com  
🔗 Código: https://github.com/seu-usuario/seu-repositorio

---

## 🧠 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar o consumo de APIs e a construção de interfaces dinâmicas com JavaScript.

A aplicação permite buscar uma cidade e visualizar:

- 🌡️ Temperatura atual (mínima e máxima)
- 💧 Umidade e velocidade do vento
- 🌧️ Probabilidade de chuva
- 🌫️ Qualidade do ar (interpretada)
- 📅 Previsão para os próximos 5 dias
- 🎨 Interface que se adapta ao clima

---

## 🛠️ Tecnologias

- JavaScript (ES6+)
- HTML5
- CSS3
- API OpenWeather

---

## ⚙️ Funcionalidades técnicas

- Consumo de múltiplas APIs simultaneamente (`Promise.all`)
- Uso de `async/await`
- Manipulação dinâmica do DOM
- Organização em módulos (API, lógica e UI)
- Tratamento de erros e validação de entrada
- Formatação de datas e dados da API

## 💡 Aprendizados

Durante o desenvolvimento, foi possível entender melhor:

- Como lidar com dados reais de APIs
- Diferença entre erro de requisição e erro de resposta
- Organização de código em projetos maiores
- Como melhorar a experiência do usuário com feedback visual
