# 🌤️ Site de Clima

Projeto simples de um site que consulta dados meteorológicos utilizando a API da OpenWeatherMap.

O usuário pode pesquisar uma cidade e visualizar algumas informações básicas sobre o clima em tempo real.

## 📋 Informações exibidas

- Cidade pesquisada
- Temperatura atual
- Temperatura máxima e mínima
- Velocidade do vento
- Umidade do ar

## 🛠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- API OpenWeatherMap

## 🎯 Objetivo do projeto

Praticar o consumo de APIs em aplicações web e trabalhar com requisições assíncronas para exibir dados em tempo real.
